3.结构体必须是4的倍数位字节！同时注意对齐字节,以最长的字节对齐，比如long字节，因此结构是最长字节的倍数！！
32bit:
char：1个字节
char*（即指针变量）: 4个字节（32位的寻址空间是2^32, 即32个bit，也就是4个字节。同理64位编译器）
short int : 2个字节
int：  4个字节
unsigned int : 4个字节
float:  4个字节
double:   8个字节
***long:   4个字节
long long:  8个字节
***unsigned long:  4个字节
64bit:
char：1个字节
char*(即指针变量): 8个字节
short int : 2个字节
int：  4个字节
unsigned int : 4个字节
float:  4个字节
double:   8个字节
***long:   8个字节
long long:  8个字节
***unsigned long:  8个字节
